Swagger

The World's Most Popular Framework for APIs.

What makes Swagger awesome?

Swagger is a simple yet powerful representation of your RESTful API. With the largest ecosystem of API tooling on the planet, thousands of developers are supporting Swagger in almost every modern programming language and deployment environment.

People like you. People at companies like Apigee, Getty Images, Intuit, LivingSocial, McKesson, Microsoft, Morningstar, and PayPal.

Some use Swagger to document their APIs. Others use it to design them. Some generate SDKs. Many contribute code.

So Swagger is awesome because people like you make it awesome. Get involved!

“Swagger gives us all the tools we need to build and manage our REST API and documentation. All the details are handled in the toolchain so we can focus on the technical complexities we have.”

— Aaron Schlesinger, Sr. member of the technical staff, PayPal
